# rpg_texto_v2

Um Roguelite Dungeon Crawler com 5 bosses únicos.

## Como jogar

1. Certifique-se de ter o Rust instalado.
2. Clone o repositório.
3. Execute 'Cargo run' no Bash.
4. Siga as instruçõxes do jogo.

## Atributos

- **Vida**: Resistência no combate
- **Força**: Dano cortante e vida.
- **Mente**: Dano mágico e cura.
- **Agilidade**: Dano perfurante e desvio.
